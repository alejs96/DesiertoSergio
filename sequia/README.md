**Banner**

Para la creación de nuestro Banner Animado utilizaremos la herramienta de Google Web Designer (https://www.google.com/webdesigner/)

Para nuestro Banner nos hemos fijado del siguiente video ( [https://www.youtube.com/watch?v=M\_KAC9pQSbs](https://www.youtube.com/watch?v=M_KAC9pQSbs))

Las imágenes las hemos obtenido de internet cambiando la configuración a reutilización no comercial.

**Edición del Banner:**

- Lo primero que haremos es crear nuestro proyecto y después creamos la edición del texto con la herramienta en forma de &quot;T&quot; que podremos encontrar en la parte izquierda del menú.
- Una vez puesto el texto nos iremos a la línea del tiempo y pulsaremos botón derecho e insertaremos un **&quot;fotograma clave&quot;.  **
- Para la aparición del texto de menos a más nos iremos a la parte de la derecha y seleccionaremos las propiedades del texto y jugaremos con posición Z de la zona de translación, rotación y escala del texto.
- Una vez realizado el paso anterior  seleccionaremos el componente llamado &quot;Area de pulsación&quot; para poder enlazar nuestro banner a nuestra página web.
- Para la aparición de las gotas de agua cambiaremos el tiempo de aparición a las veces que queramos que se repita el efecto y para ello lo encontraremos en la barra de tiempo en la parte de la izquierda.
